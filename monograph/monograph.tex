\begin{abstract}
O estudo de \emph{inteligencia artificial} comtemporâneo apresenta várias áreas de conhecimento como: processamento de linguagens naturais, programação e algoritmos genéticos, aprendizagem de máquina entre outros. Nesse trabalho, desenvolvemos um algoritmo da área de \emph{apredizagem de máquina} - mais especificamente de classificação - que tem por objetivo criar um \emph{rank} das instâncias em uma base de dados. O problema de \emph{ranking} vem ganhando importância na área de \emph{aprendizagem de máquina} nos últimos anos com diversas abordagens para uma solução, a descrita nesse trabalho consiste em usar um meta-classificador para gerar o \emph{rank} das instâncias. O programa meta-classificador foi implantado na ferramenta para \emph{aprendizagem de máquina} \emph{WEKA} assim como os resultados avaliados foram obtidos na mesma ferramenta.
\end{abstract}
